# CSV Merger – A + B → C

Simple GUI tool that merges two CSV files by matching columns **A** and **B**,  
and places the corresponding value from the second file into a new column **C**.

Match A + B (case-insensitive), add value to C from File 2
Note: Row 1 must have columns labelled A and B
Example: 
Csv 1:
A,B
Firstname,Lastname

Csv 2:
A,B
Firstname,Lastname,DOB
